Responsive site

Webside sæt op i html og css

På dette tema blev vi introduceret til kodningssprogene html og css.Jeg lærte det grundlæggende om hvordan man kan kode og at opbygge en hjemmeside. Min webside handler om kaffe og kaffekultur og er opbygget i html og css i brackets ud fra et bestemt layout diagram og wireframe. Layout diagram er en grundlæggende struktur, eller en ”opskrift”, for html som hjælper meget og kan forhindre fejl i html koden. Siden skulle være responsiv. Det betyder at siden skulle passe til alle skærme størrelser, fra desktopskærm til mobilskærm. Det er fordi tingene skal ikke bevæge sig på samme måde i mobil størrelse og desktop størrelse. Ofte skal fx billederne og teksterne være mindre og være struktureret anderledes. Det gjorde jeg med fx flex-box, grids, grid-columns og grid-rows, margin og padding, justify-content og media-queries. Med media-queries kan man definere hvad skal ske på en bestemt størrelse og passer perfekt til at lave responsivt site. Hvis der er noget jeg ville gøre anderledes på mit site, så er det farvevalg og splashbilledet, jeg synes den er lidt kedelig og mangler noget for øjet. Ellers synes jeg dette tema var spænende, lidt svært og meget nyt! Der var mange nye informationer på kort tid og meget som skulle laves.

Andre opgaver vi lavede i tema 2

På tema 2 lavede vi mange øvelser for at få bedre forståelse af html og css. Min første html og css øvelse var en kæledyr og visitkort øvelse. Øvelsen var at jeg skulle finde 2 billeder som beskriver mig og så skrive lidt tekst. Så skulle jeg style det i css, måske at tilføj nogle farver og font størrelser. Et andet eksempel på øvelse var 16 bokse øvelsen. Da skulle jeg finde mange måder til at style 16 bokse på forskellige måder i css.
